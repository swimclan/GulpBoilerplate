Gulp is Awesome

Let's bundle and minify all of this!

  1. To get started with Gulp, I need to create a gulpfile.js
  2. In our gulpfile.js we will need to require a few libraries
  3. gulp, gulp-uglify, gulp-concat, and gulp-watch
  4. Gulp is running in the tasks background
  5. Uglify is for minification and obfuscation
  6. Concat is for concatination
  7. Watch will listen for changes to your code and run tasks when changes are made
  8. Now we have to install our libraries, so for each one do something like npm install gulp --save-dev. Repeat for all other libraries.
  9. define settings. Check the gulpfile for an example
  10. After that we create our first gulp.task with a name of 'default'
  11. Gulp requires there to be a default task
